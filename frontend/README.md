# FrontEnd

Instalação

Copie o arquivo **.env.example** para **.env**

Instale as dependencias com o yarn ou npm

``yarn``

ou

``npm install``


Inicie o app

``yarn dev``

ou

``npm run dev``

**OBS**

Para o frontend funcionar verifique se o backend esta rodando na porta 3005.
Caso esteja em outra porta, altero o arquivo .env
